github常用插件参考

github高分web javascript插件，不包括node生态插件及react，vue，angular等生态插件

JavaScript 3D library（Star:32k+）：mrdoob/three.js

JavaScript 3D library，基于webgl、canvas、svg，用于AR、游戏等

moment（Star:31k+）：moment/moment

A lightweight JavaScript date library for parsing, validating, manipulating, and formatting dates.（功能过多，可供参考，不建议用于项目）

chartjs（Star:29k+）：chartjs/Chart.js

Simple HTML5 Charts using the canvas tag

underscore.js（Star:20k+）：jashkenas/underscore

JavaScript's utility _ belt

select2（Star:20k+）：select2/select2

Select2 is a jQuery based replacement for select boxes

fullPage.js（Star:18k+）：alvarotrigo/fullPage.js

全屏滚动，官网开发

Leaflet/Leaflet（Star:18k+）：Leaflet/Leaflet

JavaScript library for mobile-friendly interactive maps

axios（Star:18k+）：mzabriskie/axios

Promise based HTTP client ,ajax通用库，兼容vue，react，浏览器，node（vue，react开发必备）

PDFJS（Star:17k+）：mozilla/pdf.js

mozilla官方，纯js实现，doc文档需经服务器转换为pdf才可在线预览（在线预览pdf有多种方式，但doc文档预览必须经过pdf转换才可）

echarts（Star:17k+）：ecomfe/echarts

百度echarts canvas图表库，另外echarts-gl 3d图表库beta版：ecomfe/echarts-gl

clipboard.js（Star:17k+）：zenorocha/clipboard.js

Modern copy to clipboard. 不依赖flash

hammer.js（Star:16k+）：hammerjs/hammer.js

移动端触摸手势库

videojs（Star:16k+）：videojs/video.js

flvjs（Star:8k+）：Bilibili/flv.js

dragula（Star:15k+）：bevacqua/dragula

拖拽，demo：https://bevacqua.github.io/dragula/

PhotoSwipe（Star:13k+）：dimsemenov/PhotoSwipe

实现图片点击预览、缩放等功能，使用移动端、PC端

highlight.js（Star:9k+）：isagalaev/highlight.js

代码高亮

smartcrop.js（Star:9k+）：jwagner/smartcrop.js

图片裁切

particles.js（Star:9k+）：VincentGarreau/particles.js

knockout（Star:8k+）：knockout/knockout
